You are looking at the Agda mechanization of MetaGTLC. The main theorem
"metaevaluation is type safe" (`meta-eval-safe`) is stated and proved in
`MetaEval.agda`.

The project depends on Agda 2.7.0.1 and Agda Standard Library v2.2. It also
depends on the Abstract Binding Tree (ABT) library.

+ Agda: https://github.com/agda/agda
+ Agda stdlib: https://github.com/agda/agda-stdlib
+ ABT library: https://github.com/jsiek/abstract-binding-trees
