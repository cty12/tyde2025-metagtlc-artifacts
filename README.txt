This is the Agda development of MetaGTLC.

The project depends on Agda 2.6.4 and Agda Standard Library 1.7.3. It also depends on the Abstract Binding Tree (ABT) library.

+ Agda: https://github.com/agda/agda
+ Agda stdlib: https://github.com/agda/agda-stdlib
+ ABT library: https://github.com/jsiek/abstract-binding-trees
